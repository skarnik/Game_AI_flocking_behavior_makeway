flocking_behavior

Creates 150 flock members and 2 boid leaders.

Base class Boid contains arrive, flee, orient, move randomly, etc. methods. 

Class Flock contains flocks mechanisms such as collision avoidance, cohesion,
match velocity, make way.

Parameters can be tuned in class Flock, but don't need to be changed
for current 150 boids. 

Click some point to make boidleaders seek it, else it will move randomly.

To clear the breadcrumbs press 'd'.
